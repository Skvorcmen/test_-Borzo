# BORZO Script Navigator

Внутренний навигатор скриптов для управления скриптами продаж и обработки возражений. Система позволяет менеджерам просматривать скрипты и предлагать изменения, а руководителям — модерировать и утверждать предложения.

## 🌐 Демо

Приложение доступно по адресу: [[https://test--borzo.onrender.com](https://test--borzo.onrender.com)](https://skvorcmen.pythonanywhere.com/)

## 👥 Роли и доступ

| Роль | Пароль | Возможности |
|------|--------|-------------|
| **Менеджер** | `` | Просмотр скриптов, предложение изменений, создание новых блоков/разделов |
| **Руководитель** | `` | Полный доступ: редактирование, одобрение/отклонение предложений, экспорт данных |

## 🚀 Функциональность

### Для менеджера
- 📖 Просмотр магистральных скриптов и разделов
- ✏️ Предложение изменений существующих сообщений
- ➕ Создание предложений на новые блоки, элементы и разделы
- 📋 Копирование текстов сообщений в буфер обмена
- 🔍 Управление своими предложениями

### Для руководителя
- ✍️ Редактирование любых блоков и сообщений
- ✅ Одобрение предложений (с возможностью корректировки)
- ❌ Отклонение предложений с комментариями
- 🗑️ Удаление блоков, разделов, сообщений
- 📊 Экспорт базы в текстовый файл
- 💾 Скачивание проекта в формате JSON
- 🎯 Drag & Drop для сортировки

## 🛠️ Технологии

- **Backend**: Python 3 + Flask
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Хранение данных**: JSON файл (data.json)
- **Деплой**: pythonanywhere.com
